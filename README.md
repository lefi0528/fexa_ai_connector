# Fexa AI Connector for PrestaShop

**Official MCP Server for Fexa AI Integration**

Connect your PrestaShop store effortlessly to [Fexa AI](https://fexaai.com), the advanced SEO & Automation platform.

## Features

- **Universal Compatibility**: Works on PrestaShop 1.7 (PHP 7.4) and PrestaShop 8/9 (PHP 8.1+).
- **Plug & Play**: Simple installation, zero complex configuration.
- **Secure**: Uses MCP (Model Context Protocol) with API Key authentication.

## Installation

1. Download the latest release `.zip`.
2. Go to your PrestaShop Admin Panel > Modules > Module Manager.
3. Click "Upload a module" and select the zip file.
4. Configure the module to get your API Key.
5. Paste the API Key in your Fexa AI Dashboard.

## Requirements

- PrestaShop >= 1.7.0.0
- PHP >= 7.4
